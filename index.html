Statistical Summary -

This section includes all statistics description corresponding to the selected model. It includes:

  1. The name of the model
  2. The list of independent and dependent variables with a short description
  3. A table with the model result starting with the dependent variable (constant), and independent variables. Table includes all coefficient required to interpret the results
  4. A table that provide the determination coefficients
  5. Graphics representing each case scenario for the model
  6. A map displaying the investigated area for the type of properties

Description -

This section includes the verbiage describing the interpretation of the tables corresponding to the selected model.
